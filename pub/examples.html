Examples

Default Configuration


   DriftCarousel.config = {
      transitionTimeout: 5000,
      transitionDuration: 1000,
      random: false,
      brightness: 1,
      opacity: 1,
      captions: false,
      indicators: true,
      arrows: true,
   };
The only things Drift requires are a selector and an array of images. 

   const defaultCarousel = new DriftCarousel("#default-carousel", exampleImages);
   defaultCarousel.renderCarousel(0);
#default-carousel

Custom Configuration

Any of the default configuration options can be overwritten by passing in a configuration object. 

   const landscapeCarouselConfig = {
      transitionTimeout: 3000,
      transitionDuration: 2000,
      random: true,
      opacity: 0.5,
   };

   const landscapeCarousel = new DriftCarousel(
      "#landscape-carousel",
      exampleImages,
      landscapeCarouselConfig
   );
   landscapeCarousel.renderCarousel(0);
#landscape-carousel

Full Configuration:

  • Transition Timeout: 3s
  • Transition Duration: 2s
  • Random: true
  • Brightness: 1 (default)
  • Opacity: 0.5
  • Captions: false (default)
  • Indicators: true (default)
  • Arrows: true (default)

Image Orientation & Captions

Images are automatically sized to fully cover the carousel without warping.
Captions can contain HTML elements to allow for more flexibility in styling. 

   /** Left Carousel *************************************/

   const portraitCarouselLeftConfig = {
      transitionTimeout: 3000,
      random: true,
      captions: true,
      arrows: false,
   };
   const portraitCarouselLeft = new DriftCarousel(
      "#portrait-carousel-left",
      exampleImages,
      portraitCarouselLeftConfig,
      captions
   );
   portraitCarouselLeft.renderCarousel(0);



   /** Right Carousel *************************************/

   const portraitCarouselRightConfig = {
      transitionTimeout: 3000,
      brightness: 0.55,
      indicators: false,
   };
   const portraitCarouselRight = new DriftCarousel(
      "#portrait-carousel-right",
      exampleImages,
      portraitCarouselRightConfig
   );
   portraitCarouselRight.renderCarousel(0);
#portrait-carousel-left

  • Transition Timeout: 3s
  • Transition Duration: 1s (default)
  • Random: true
  • Brightness: 1 (default)
  • Opacity: 1 (default)
  • Captions: true
  • Indicators: true (default)
  • Arrows: false
