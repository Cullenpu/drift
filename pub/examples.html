Sample Usage

#default-carousel

The only things Drift requires are a <div> selector and an array of images. Note that the selected <div> must have a height and width; this will be the height and width of the carousel. It can also contain other HTML elements, as seen in the examples below. If so, Drift will act as a background rather than a standalone carousel. The best part? The code remains the same for either situation! 


  const exampleImages = [
    "res/example/img1.jpg",
    "res/example/img2.jpg",
    "res/example/img3.jpg",
    "res/example/img4.jpg",
    "res/example/img5.jpg",
  ];

  const defaultCarousel = new Drift("#default-carousel", exampleImages);
  defaultCarousel.renderCarousel(0);

Default Configuration

Drift comes with a default configuration. These options were chosen to cover a wide range of carousel usage. Don't worry if there's an option you don't like - any of these options can be changed. 


  Drift.config = {
    transitionTimeout: 5000,
    transitionDuration: 1000,
    random: false,
    brightness: 1,
    opacity: 1,
    captions: null,
    captionSize: "10px",
    indicators: true,
    arrows: true,
  };

Custom Configuration

Any of the default configuration options can be overwritten by passing in a configuration object. You can change as little or as many of the options as you want. The rest will keep their default values.

#landscape-carousel

Full Configuration:

  • Transition Timeout: 3s
  • Transition Duration: 2s
  • Random: true
  • Brightness: 1 (default)
  • Opacity: 0.5
  • Captions: null (default)
  • Caption Size: 10px (default)
  • Indicators: true (default)
  • Arrows: true (default)

   const landscapeCarouselConfig = {
      transitionTimeout: 3000,
      transitionDuration: 2000,
      random: true,
      opacity: 0.5,
   };

   const landscapeCarousel = new Drift(
      "#landscape-carousel",
      exampleImages,
      landscapeCarouselConfig
   );
   landscapeCarousel.renderCarousel(0);

Image Orientation & Captions

Drift offers simplicity and figures things out for you - images are automatically resized to fully cover the carousel without warping no matter the carousel orientation.

Captions are also intuitive. They can contain HTML elements and tags to allow for more flexibility in styling.

#portrait-carousel-left

Full Configuration:

  • Transition Timeout: 3s
  • Transition Duration: 1s (default)
  • Random: true
  • Brightness: 1 (default)
  • Opacity: 1 (default)
  • Captions: captions
  • Caption Size: 10px (default)
  • Indicators: true (default)
  • Arrows: false
#portrait-carousel-right

Full Configuration:

  • Transition Timeout: 3s
  • Transition Duration: 1s (default)
  • Random: false (default)
  • Brightness: 0.55
  • Opacity: 1 (default)
  • Captions: null (default)
  • Caption Size: 10px (default)
  • Indicators: false
  • Arrows: true (default)

   /** Left Carousel *************************************/

   const portraitCarouselLeftConfig = {
      transitionTimeout: 3000,
      random: true,
      captions: captions,  // Array of captions
      arrows: false,
   };
   const portraitCarouselLeft = new Drift(
      "#portrait-carousel-left",
      exampleImages,
      portraitCarouselLeftConfig
   );
   portraitCarouselLeft.renderCarousel(0);



   /** Right Carousel *************************************/

   const portraitCarouselRightConfig = {
      transitionTimeout: 3000,
      brightness: 0.55,
      indicators: false,
   };
   const portraitCarouselRight = new Drift(
      "#portrait-carousel-right",
      exampleImages,
      portraitCarouselRightConfig
   );
   portraitCarouselRight.renderCarousel(0);
