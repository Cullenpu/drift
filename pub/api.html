API

Constructor

An instance of Drift must be created before it is used.

const drift = Drift(selector, images, config)
Property Type Description
selector String

A selector of the root element to mount Drift.
images Array

The list of image paths for the carousel to render.
config (optional) Object

Optional configuration object to override default configuration.

Configuration Object

Configuration options can be set by passing in a configuration object with the desired properties. Only the properties to be changed need to be included in the object.

Property Type Default Description
transitionTimeout Number 5000

The time in milliseconds each image is shown before switching to the next image.

transitionDuration Number 1000

The time in milliseconds each carousel transition takes.
random Boolean false

If the carousel randomly picks images when switching. If set to false, the carousel advances in the order of the images array used in the constructor.

brightness Number 1

The carousel image brightness, between 0 and 1.
opacity Number 1

The carousel image opacity, between 0 and 1.
captions Array null

An array of strings corresponding to the list of images passed in the constructor. The length of the array must be the same as the number of images in the carousel. HTML can also be used in the caption strings for greater flexibility such as links and text formatting.

indicators Boolean true

If the bottom image indicators are shown.
arrows Boolean true

If the arrows are shown.

API Methods

.renderCarousel(i)

This must be called to render the carousel. It will start with the image at index i.


Drift provides an alternate way to set the configuration through the given API methods.

.setTransitionTimeout(transitionTimeout)

Sets the transition timeout configuration to transitionTimeout.

.setTransitionDuration(transitionDuration)

Sets the transition duration configuration to transitionDuration.

.setRandom(toggle)

Sets the random ordering configuration to toggle.

.setBrightness(brightness)

Sets the image brightness configuration to brightness.

.setOpacity(opacity)

Sets the image opacity configuration to opacity.

.setCaptions(captions)

Sets the captions to the captions array if specified or removes them altogether if null is passed. Like before, if captions are being set, the length of the array must be equal to the number of images in the carousel.

.setIndicators(toggle)

Sets the indicators configuration to toggle

.setArrows(toggle)

Sets the arrows configuration to toggle.
